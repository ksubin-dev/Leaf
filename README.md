# Leaf

| Category | Tech | 적용 목적 |
|---|---|---|
| UI | Jetpack Compose (Material3) | 복잡한 입력 UI를 선언형으로 안정 구현 |
| Architecture | MVVM + UseCase | 책임 분리로 확장성·유지보수성 확보 |
| Async / State | Coroutines + Flow (StateFlow/SharedFlow) | 상태·이벤트를 일관된 방식으로 관리 |
| DI | Hilt | 의존성 주입 자동화로 결합도 감소 |
| Backend / Cloud | Firebase (Auth / DB / Storage) | 인증·데이터·이미지 저장 빠르게 구축 |
| Background | WorkManager | 앱 이탈에도 저장 작업 안정 처리 |
| On-device AI | ML Kit Text Recognition (OCR) | 라벨 텍스트 인식으로 입력 부담 감소 |
| Tooling | GitHub, Figma | 버전 관리 및 UI 설계 정리 |
